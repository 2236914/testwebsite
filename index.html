Generation of Computers

Computer generation describes the unique advancements in computer technology across time. Circuits, which are electrical paths, were created to carry out the counting in 1946. It took the place of the counting gears and other mechanical components seen in earlier computers. The circuits got smaller and more sophisticated with each successive generation, compared to the circuits of the prior generation. The miniaturization aided in boosting computers' speed, memory, and power. The following list of five computer generations includes:

1st Generation: 1946-1959

Computers of the first generation were slow, large, and expensive. Vacuum tubes served as the foundational elements of the CPU and memory in these computers. These computers were mostly dependent on punch cards and batch operating systems. Magnetic tape and paper tape were utilized as output and input devices in this generation. Several of the well-known first-generation computers include;

1. ENIAC ( Electronic Numerical Integrator and Computer)
2. EDVAC ( Electronic Discrete Variable Automatic Computer)
3. UNIVACI( Universal Automatic Computer)
4. IBM-701
5. IBM-650

2nd Generation: 1959-1965

Transistor computers were faster than first-generation computers because they used cheaper, more energy-efficient, and smaller transistors. Magnetic cores were employed as the primary memory in this generation, with magnetic discs and tapes serving as the secondary storage. These computers used batch processing, multiprogramming operating systems, assembly language, and programming languages like COBOL and FORTRAN. Several of the well-known second-generation computers include;

1. IBM 1620
2. IBM 7094
3. CDC 1604
4. CDC 3600
5. 1108

3rd Generation: 1966-1670

Transistors were replaced by integrated circuits (ICs) in computers of the third generation. A single integrated circuit (IC) can house a sizable number of transistors, increasing a computer's power while decreasing its price. Additionally, computers have grown more dependable, effective, and compact. Several of the well-known third-generation computers are;

1. IBM-360 series
2. Honeywell-6000 series
3. PDP(Personal Data Processor)
4. IBM-370/168
5. TDC-316

4th Generation: 1971-1980

Very large scale integrated (VLSI) circuits, a chip with millions of transistors and other circuit components, were utilized in the fourth generation of computers. These chips improved the size, performance, speed, and affordability of computers in this generation. These computers used a distributed, real-time, and time-sharing operating system. This generation also used programming languages like C, C++, and DBASE. Some popular fourth-generation computers include;

1. DEC 10
2. STAR 1000
3. PDP 11
4. CRAY-1(Super Computer)
5. CRAY-X-MP(Super Computer)

5th Generation: 1980-Present

VLSI was replaced with ULSI technology in computers of the fifth generation (Ultra Large Scale Integration). The manufacturing of microprocessor chips with ten million electronic components was made practicable. AI (Artificial Intelligence) software and parallel processing hardware were used in computers of this generation. This generation used programming languages like C, C++, Java,.Net, etc. Some of the popular fifth generation computers are;

1. Desktop
2. Laptop
3. NoteBook
4. UltraBook
5. ChromeBook

SOFTWARE

Software. It is a collection of computer programs along with supporting files and information. Hardware, on the other hand, is what the system is made of and what actually does the work.



History of Software

First Theory of Software

Although Ada Lovelace's programs for Charles Babbage's Analytical Engine in the 19th century are frequently credited as founding the field, the mathematician's work remained purely theoretical because Babbage's computer could not be constructed using the technology of the time. In 1935, Alan Turing is credited with developing the first theory of software, which gave rise to the academic disciplines of computer science and software engineering.

Development of Modern Programming Language

Directly created in binary code, the initial generation of software for early stored-program digital computers in the late 1940s was often written for mainframe computers. Beginning with assembly language and moving through functional programming and object-oriented programming paradigms, the later development of modern programming languages along with the advancement of the home computer would greatly expand the range and breadth of software that was readily accessible.

Binary Code

The first computer programmers used binary code to instruct computers to perform tasks. ForTRAN was developed by John Backus at IBM in the 1950s. COBOL was conceived of when Mary K. Hawes convened a meeting (which included Grace Hopper) in 1959. Jean E. Sammet's book Programming Languages became influential text.

PROGRAMMING LANGUAGE

Programming Language. It is a system of notation for writing computer programs. Most programming languages are text-based formal languages, but they may also be graphical. They are a kind of computer language.


Timeline of Programming Language

1840s

1843: Ada Lovelace’s machine algorithm - For Charles Babbage's Difference Machine, Ada Lovelace creates the first machine algorithm, which serves as the basis for all programming languages.

1940s

1944-45: Plankalkül - It was created by Konrad Zuse and the first real programming language.

1949: Assembly Language - The Electronic Delay Storage Automatic Calculator was written in Assembly (EDSAC). Low-level programming languages like assembly language made machine code easier to understand.

1949: Shortcode - John McCauley first proposed Shortcode (or Short-order code) as a High-Level Language (HLL). William Schmitt was the one who put it into practice for the UNIVAC in 1950 as well as the BINAC computer that same year.

1950s

1952: Autocode - Autocode, the first-ever compiled language to be implemented, was created by Alick Glennie for the Mark 1 computer at the University of Manchester.

1957: FORTRAN - John Backus is credited with developing FORmula TRANslation, also known as FORTRAN, which is the oldest programming language still in use. The programming language was developed for complex calculations in science, math, and statistics.

1958: ALGOL (Algorithmic Language) - A joint committee of American and European computer scientists developed the algorithmic language, or ALGOL.

1958: LISP (List Processor) - John McCarthy created the list processor, or LISP, at the Massachusetts Institute of Technology (MIT). LISP, one of the oldest programming languages still in use today and originally intended for artificial intelligence, can be used in place of Ruby or Python.

1959: COBOL (Common Business Oriented Language) - The programming language used by many credit card processors, ATMs, phone and cell calls, hospital signals, and traffic signal systems is called Common Business Oriented Language (COBOL). Dr. Grace Murray Hopper oversaw the language's creation, and it was created to be compatible with all makes and models of computers. The main applications of COBOL today are gamification and banking systems.

1960s

1964: BASIC (Beginner's All-Purpose Symbolic Instruction Code) - The Beginners All-Purpose Symbolic Instruction Code, also known as BASIC, was created by a group of Dartmouth College students. The first commercially successful offering from Microsoft was the language that Bill Gates and Paul Allen further developed.

1970s

1970s: PASCAL - Niklaus Wirth created the programming language in honor of the French mathematician Blaise Pascal. It was created as a tool for teaching computer programming. It was favored by Apple in the early years of the company because of its simplicity and strength.

1972: Smalltalk - Computer programmers could alter code on the fly thanks to Smalltalk, which Alan Kay, Adele Goldberg, and Dan Ingalls created at the Xerox Palo Alto Research Center.

1972: C - Dennis Ritchie created the software at the Bell Telephone Laboratories for the Unix operating system. Because it was based on an earlier language called "B," it was given the name "C." Many of the most popular languages used today, including C#, Java, JavaScript, Perl, PHP, and Python, are derivations of C.

1972: SQL (SEQUEL at the time) - Raymond Boyce and Donald Chamberlain, two IBM researchers, created SQL for the first time. Information saved in databases can be viewed and modified using SEQUEL. Structured Query Language, or SQL, is the name of the language used today.

1980s

1980/81: Ada - Jean Ichbiah created the original version of Ada for the US Department of Defense. Ada is a structured, statically typed, imperative, wide-spectrum, and object-oriented high-level programming language that bears the name of the mathematician

1983: C++ - At Bell Labs, Bjarne Stroustrup improved the C language; C++ is an extension of C with new features including classes, virtual functions, and templates. Since 1986, it has consistently ranked in the top 10 programming languages, and in 2003, it was inducted into the Hall of Fame. Game engines, Adobe Photoshop, Microsoft Office, and other high-performance software all employ C++.

1983: Objective-C - The primary programming language used to create software for Apple's operating systems, macOS and iOS, was created by Brad Cox and Tom Love.

1987: Perl - It is a general-purpose, high-level programming language that was developed by Larry Wall. Originally intended as a text editing scripting language, it is now widely used for a variety of tasks including CGI, database applications, system administration, network programming, and graphic programming.

1990s

1990: Haskell - A general-purpose programming language called Haskell is named for the American mathematician and logician Haskell Brooks Curry. It is a totally functional programming language, which means that math is at its core.

1991: Python - It is one of the most widely used programming languages in the world today and created by Guido Van Rossum named after the British comedy group "Monty Python." It is a high-level, general-purpose programming language designed to support a range of programming paradigms and be enjoyable to use.

1991: Visual Basic - Microsoft created Visual Basic, which enables programmers to use a drag-and-drop method to choose and modify already-selected sections of code using a graphical user interface (GUI).

1993: Ruby - Ruby is an interpreted high-level programming language that was created by Yukihiro Matsumoto, a teaching tongue that was influenced by, Perl, Ada, Lisp, and Smalltalk. Some well-known companies that use Ruby include Twitter, Hulu, and Groupon.

1995: Java - James Gosling developed Java as a general-purpose, high-level language for an interactive TV project. It is consistently ranked among the top programming languages in the world and provides cross-platform features. Java is used in all kinds of devices.

1995: JavaScript - Brendan Eich developed the JavaScript programming language, which is largely used for desktop widgets, PDF documents, and dynamic web development. Nearly all significant websites utilize JavaScript. Some well-known examples include Mozilla Firefox, Adobe Photoshop, and Gmail.

2000s

2000: C# - It is based on C++ and is very similar to Java, was created at Microsoft in the hopes of fusing the computing power of C++ with the ease of Visual Basic. Almost all Microsoft products use the language, which is primarily used for creating desktop applications.

2003: Scala - A programming language created by Martin Odersky that blends organized object-oriented programming with mathematical functional programming. Due of its Java interoperability, Scala is useful for Android development.

2003: Groovy - It was created by James Strachan and Bob McWhirter and is based on Java. The language's conciseness and ease of learning increase productivity.

2009: Go - Google created Go to solve problems brought on by complicated software systems. Go has been popular among some of the biggest tech businesses in the world, like Google, Uber, Twitch, and Dropbox, because to its straightforward and cutting-edge framework.

2014: Swift - A programming language created by Apple to replace C, C++, and Objective-C, was designed to be simpler and offer less room for error than the aforementioned languages. Due to its adaptability, Swift is suitable for desktop, mobile, and cloud applications.

Most Pragramming Language Today

Computer programming has advanced beyond all expectations and now powers much of the technology we use every day. Because of this, we require qualified specialists to implement and manage the computer languages that run our modern society.

According to the DevSkiller 2020 IT Skills Report, the report contains data from over 213,000 coding tests sent across 143 countries by companies looking to hire specialized IT professionals. For the eighth year in a row, JavaScript was found to be the most popular programming language.
